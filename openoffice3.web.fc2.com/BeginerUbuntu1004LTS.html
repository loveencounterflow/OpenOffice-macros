初心者Memo / LibreOffice(Ubuntu編)



 1. Ubuntu10.04LTSへのInstall

    1)既存のOfficeをUn-Install

    sudo apt-get remove openoffice*.*


    2)既存のPPAの削除

    sudo add-apt-repository ppa:libreoffice/ppa


    3)LibreOfficeのInstall

    sudo apt-get update

    sudo apt-get install libreoffice


    4)LibreOfficeの日本語化

    「Synapticパッケージ･マネージャ」にて以下のPakageを追加
       1) Libreoffice-l10n-ja
       2) Libreoffice-gtk( <= 文字化け対策)


 2. KDE4上の文字化け対策

    1)Defualtの日本語Font設定を変更

    「ツール」=>「オプション」=>「LibreOffice」=>「フォントの種類」=>「フォントの種類(D)」の項目を
      「Droid Sans」=>「Umeplus Gothic」等の日本語フォント
    に変更する。

macro-)Command LineからMacro実行
     OpenOffice.org / LibreOffice Basicの場合( Windows )

     [ 条件 ]
     ・PC固有の「マイマクロ」下に指定のmacroがある事。
      1) 「soffice.exeの格納Folderに移動( Pathがついていれば不要 )
      2) >soffice macro:///[Library Name].[Module Name].[Macro Name]

      Library Name : Standard
      Module Name : Module1
      Macro Name : Main

     の場合   >soffice macro:///Standard.Module1.Main

     でOpenOffice.org / LiberOfficeが起動し、所定のMacroを実行する。
     Macro実行後はOpenOffice.org / LibreOfficeは自動で終了する。
